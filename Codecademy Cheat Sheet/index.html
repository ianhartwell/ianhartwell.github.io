HTML Basics

Basic Elements

Element Description
<!DOCTYPE html> Document type declaration.
<html> The root element of an HTML page.
<head> Contains meta information about the HTML page.
<title> Specifies the title of the HTML page. Displayed in the browser title bar or page tab.
<body> The document body and container for all of the pages visible content.
<div> Non-semantic element. Container for other HTML elements.
<h1> Heading element. One of six elements (h1, h2, h3. h4, h5, h6) with descending order of importance.
<p> Paragraph element.
<a> Link element with destination specified using href.

Semantic Elements

Element Description
<header> A container for introductory content or a set of navigational links.
<nav> A set of navigation links.
<main> Specifies the main content of a documente.
<section> A thematic grouping of content, typically with a heading.
<article> Independent, self-contained content.
<figure> Specifies self-contained content, like illustrations, diagrams, photos, code listings, etc.
<figcaption> A caption for a figure element.
<aside> Content aside from the page content.
<footer> Defines a footer for a document or section.

HTML Table Reference

Table Tags

Tag Name Description
<table> Table The wrapper element for all HTML tables.
<th> Table Head The set of rows defining the column headers in a table.
<tb> Table Body The set of rows containing actual table data.
<tr> Table Row The table row container.
<td> Table Data The table row container.
<tf> Table Foot The set of rows defining the footer in a table.

Table Attributes

Attribute Name Description
colspan Column Span Defines how many columns a td element should span.
rowspan Row Span Defines how many rows a td element should span.
